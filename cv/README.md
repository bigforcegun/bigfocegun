## Compile and analyse

Follow [JsonResume](https://jsonresume.org/) docs

Or just run

```shell script
git clone https://github.com/bigforcegun/bigfocegun.git && cd ./bigforcegun/cv
npm install
npm run serve
npm run analyze
```
